    On this page you will find some examples of web-application projects completed while working through FreeCodeCamp's full stack developer curriculum.

    Project 1

    Drum Machine

    Drum Machine

    This is a web-application for simulating a drum machine. The user can eithe ruse keyboard shortcuts to play selected sounds or manually click the "pads" using their mouse. A sound is played when one of the "pads" is hit, the sound name is displayed in the bottom window.

    Project 2

    Bubble Sort Visualizer

    Bubble Sort Visualizer

    This is a web-application for simulating the bubble sort algorithm. The app generates a list of 5 numbers and then sorts the list. At each step it shows the action it is performing by outlining the numbers attempting to be swapped in a dotted red line. The program can loop by pressing the generate array button resetting the view window.

    Project 3

    MarkDown to HTML Converter

    Markdown to HTML Converter

    This project takes user input for certain markdown syntax and converts them to both raw HTML code and an html preview. Accepted inputs include: Headings 1,2 & 3, Bold text, Italic text, Images, linkes and quotes.
