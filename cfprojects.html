    • Computational Fabrication

    This page is a general overview of the projects, more details and project files can be found here: Github Repository Computational Fabrication

    These projects showcase my work in a computational fabrications course, exploring the intersection of design, coding, and 3D printing. Each project was developed in Grasshopper-Rhino using Python and brought to life with a Creality Ender 3 3-D printer.

    Project 1

    Parametric Vessels

    For this project, we were tasked with developing a program to generate 3D parametric vessels. The program constructs these vessels by stacking layers of polygons on top of one another. Each layer is customizable with an adjustable number of sides, radius, and tilt angle, enabling endless design possibilities.

    Project 2

    Data Physicalization

    For this project, we were tasked with creating objects from selected datasets. My first object was derived from MIDI data of Johann Sebastian Bach's Partita No. 3 for solo violin. I created a python program to read a midi file and Each note was assigned a numeric value from 0 to 127. Groups of four notes were averaged and saved to a CSV file, and I selected 100 data points from these averages to form the basis of a vessel. The second object was generated from a bitmap representation of a topographic map of the Santa Fe ski area. Each pixel's brightness was used to define elevation values, constructing the topography row by row.

    Project 3

    Computational Ukulele

    For this project, I aimed to construct a customizable ukulele by applying the skills learned throughout the course. The body of the instrument is defined by two curves in Rhino that shape its sides and top walls. Due to technical constraints, the neck of the instrument is a 3D model rather than being computationally generated.
